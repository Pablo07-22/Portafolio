## 📂 Proyectos

### ☕ Proyecto de Caffe

Sistema para la gestión de una cafetería que permite:

- Registrar productos.
- Actualizar productos.
- Eliminar productos.
- Facturar pedidos.
- Consultar productos disponibles.

Este proyecto implementa operaciones CRUD (Crear, Leer, Actualizar y Eliminar) y funciona como una carta digital para la realización de pedidos.

### 🔧 Gestión de Herramientas

Sistema para la administración y préstamo de herramientas.

Características:

- Gestión de administradores.
- Registro de vecinos.
- Control de préstamos.
- Inventario de herramientas.
- Operaciones CRUD completas.
